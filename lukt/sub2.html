    재료 - 룩트 요거트, 허니스틱, 과일스틱, 그래놀라

    1. 룩트 마일드 요거트에 넣고 싶은 잼이나 꿀을 넣어요
    2. 잘 섞은 후 유산지 위에 넓게 펴줘요
    3. 원하는 토핑을 얹어요
    4. 냉동실에서 꽁꽁 얼려준 후 맛있게 먹어요

  • 한라봉&무화과 오픈샌드위치

    재료 - 마일드 요거트, 통밀빵, 한라봉, 무화과, 허니스틱

    1. 프라이팬에 버터를 녹인 후 준비한 통밀빵을 노릇하게 구워요
    2. 마일드 요거트를 구운 빵 위에 발라요
    3. 한라봉을 자른 후 요거트가 발린 빵 위에 얹은 후 그래놀라나 피스타치오를 뿌려줘요
    4. 마지막으로 취향껏 꿀을 뿌리면 완성!

  • 요거트 바크 레시피

    재료- 마일드 요거트, 그래놀라, 허니스틱, 과일토핑

    1. 마일드 요거트와 꿀을 섞어주세요
    2. 꿀을 섞은 요거트를 사각형의 용기에 펴서 담아주세요
    3. 그래놀라와 과일을 예쁘게 올려주세요
    4. 용기에 뚜껑을 닫아 냉동실에서 5시간 얼려주세요

  • 피치&얼그레이 마리에이드 레시피

    재료 - 황도&복숭아, 얼그레이 잎, 띠크 요거트

    1. 황도 또는 복숭아를 먹기 좋은 크기로 준비해주세요
    2. 얼그레이 잎을 적당량 넣어 섞은 후 향이 베어들게 10분 정도 냉장고에서 숙성시켜주세요
    3. 꾸덕한 요거트를 올려서 먹어요

  • 약과 샌드 레시피

    재료 - 띠크 요거트, 약과, 그래놀라

    1. 준비한 약과에 꾸덕한 띠크 요거트를 발라요
    2. 버거처럼 윗면을 덮어줘요
    3. 반복해서 여러 개 만들어요
    4. 인절미 소보로 그래놀라를 취향껏 뿌려줘요

  • 그릭토마토 레시피

    재료 - 토마토, 띠크 요거트, 허니스틱, 그래놀라

    1. 깨끗하게 씻은 토마토를 십자모양으로 칼집을 내준 후 끓는 물에 살짝 데쳐주세요
    2. 숟가락을 이용해 중간 심지와 속을 살짝 파줘요
    3. 꾸덕한 그릭 요거트로 속을 꽉꽉 채워주세요
    4. 그래놀라를 토핑하고 꿀을 휘리릭 돌려주세요
